# Makefile

## Definition
A Makefile is used to automate compilation and execution tasks.

---

## Simple Makefile Example

### Makefile
```makefile
all:
	echo "Building Project"

run:
	echo "Running Project"

clean:
	echo "Cleaning Files"
```

---

## Run Build

```bash
make
```

### Output
```bash
Building Project
```

---

## Run Program

```bash
make run
```

### Output
```bash
Running Project
```

---

## Clean Files

```bash
make clean
```

### Output
```bash
Cleaning Files
```
